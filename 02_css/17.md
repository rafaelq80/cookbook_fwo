**Linha 06:** A propriedade `display: grid` transforma o container em uma grade composta por linhas e colunas. A opﾃｧﾃ｣o `grid` define que o contﾃｪiner se comportarﾃ｡ como um elemento de nﾃｭvel de bloco, ou seja, o elemento container ocuparﾃ｡ toda a largura disponﾃｭvel e serﾃ｡ organizado como uma tabela, composta por linhas e colunas.

Como no projeto Portfﾃｳlio, utilizaremos 2 colunas com o mesmo tamanho, onde na primeira posicionaremos as informaﾃｧﾃｵes sobre as Redes Sociais e na segunda posicionaremos o Formulﾃ｡rio de Contato, podemos utilizar a funﾃｧﾃ｣o CSS **repeat()**. A funﾃｧﾃ｣o repeat possui 2 parﾃ｢metros:

- **Numero de Repetiﾃｧﾃｵes:** Neste parﾃ｢metro, indicamos quantas colunas serﾃ｣o criadas. A opﾃｧﾃ｣o **auto-fill** criarﾃ｡ quantas colunas forem necessﾃ｡rias atﾃｩ preencher todo o container.
- **Tamanho da Coluna:** Neste parﾃ｢metro definimos o tamanho de cada coluna, que serﾃ｡ repetida. Definimos o tamanho 518px (inferior a metade do tamanho total do container - 1120px) para adicionar espaﾃｧamentos entre os elementos que serﾃ｣o posicionados no grid.

**Linha 07:** A propriedade `grid-template-columns` especifica o nﾃｺmero e a largura das colunas do layout de grade. Os valores podem ser definidos por uma lista, separada por espaﾃｧos, onde cada valor especifica o tamanho da respectiva coluna.

**Linha 10:** A propriedade `gap` define o espaﾃｧamento entre as linhas (row-gap) e o espaﾃｧamento entre as colunas (column-gap) de uma ﾃｺnica vez.

**Linha 13:** Definimos o seletor de classe `.contato_container h4`, que aplicarﾃ｡ estilos em todas as tags `<h4>` (titulo 04) dos elementos estilizados pela classe `contato_container`.
